# BOOK
#  Сайт по обмену книгами

Веб-приложение для организации обмена книгами между пользователями без денежных затрат.

##  Технологии
- **HTML5** — разметка
- **CSS3** — стили (Flexbox, Grid)
- **JavaScript** — интерактивность
- **PHP 8.x** — серверная логика
- **JSON** — хранение данных (вместо MySQL)

##  Как запустить проект

1. Установите локальный сервер (например, **XAMPP** или **OpenServer**).
2. Скопируйте папку проекта в директорию сервера (например, `htdocs` или `domains`).
3. Запустите сервер.
4. Откройте в браузере: `http://localhost/book/`.
